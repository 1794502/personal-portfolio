---

## 🌍 Deployment

This project is deployed using **GitHub Pages**.

Steps used:

1. Push project to GitHub.
2. Go to Repository → Settings → Pages.
3. Select branch: `main`.
4. Select folder: `/ (root)`.
5. Save and wait for deployment.
